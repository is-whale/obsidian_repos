### gtsam库安装

  

1.安装依赖

```Shell
sudo apt-get install libboost-all-dev
sudo apt-get install cmake
sudo apt-get install libtbb-dev
```

2.手动编译安装

```Shell
git clone https://bitbucket.org/gtborg/gtsam.git
cd gtsam/
mkdir build &&cd build
cmake ..
make check
sudo make install
```

### asio依赖项

```Plain
sudo apt-get update

```
